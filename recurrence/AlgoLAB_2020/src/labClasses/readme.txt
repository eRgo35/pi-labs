Implement your solutions in algoA, algoB, algoC methods of FractalAlgorithms class 
Remember of input parameters.
Available commands for Turtle:
-> forward(int length )
-> back(int length )
-> right( int angle )
-> left( int angle )
Angle can be negative

Tu run and test your solutions use RunFractalAlgos class
REMEMBER JAVA IS CASE SENSITIVE 

In case of problems with project after import try this:

1) Right click on the project’s name, choose Properties, 
    check if Java (JDK) version is correct in Sources and Libraries categories. 
    If not, choose JDK version that is available on your machine.
2) Right click on the project’s name, choose Clean and Build option.
